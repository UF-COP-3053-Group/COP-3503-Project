This file contains a list of known errors, their causes, and potential fixes

Known Calculation Errors:

pi / 2:
	Returns: 3 / 2
	Cause: The Rational.simplify() method only supporting integers
	Fix: Allow the simplify method to work on more than just integers



General Program Errors:
Memory Leaks, most notably in Expression where setting new nodes can't delete the old nodes reliably.
